Game 1: The Red Block Grid 🟥
Domain: Spatial Logic & Forward Planning 

Overview: A spatial puzzle, this module is a high-fidelity telemetry sensor. Instead of a binary "pass/fail," it mathematically maps the player's internal problem-solving process.

Key Telemetry Tracked (The Inputs):

First-Move Latency: Measures mental simulation before acting.

Path Consistency: Tracks commitment to a planned route.

Excess Moves: Indicates reliance on brute-force trial-and-error.
